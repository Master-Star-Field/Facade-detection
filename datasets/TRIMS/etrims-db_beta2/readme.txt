06-Mar-2009, eTRIMS Image Database beta 2
-----------------------------------------

This is the beta 2 release of the eTRIMS Image Database (http://www.ipb.uni-bonn.de/projects/etrims_db/) with 60 annotated images. We are preparing larger public database release that is to be published on the web page. If you wish to be informed, please send email to Filip Korc (filip.korc@uni-bonn.de).
